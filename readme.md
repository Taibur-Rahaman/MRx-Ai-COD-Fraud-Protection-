# MRx AI - COD Fraud Protection

Smart COD fraud protection system for Bangladesh e-commerce. Reduces fake COD orders by 80-90% using risk scoring, phone tracking, and behavior analysis.

## Features

- ✅ Risk Scoring Engine
- ✅ Phone Reputation Tracking
- ✅ IP Address Tracking
- ✅ Device Fingerprinting
- ✅ Behavior Analysis
- ✅ Dokan Marketplace Integration
- ✅ Courier Feedback System
- ✅ Facebook Conversions API Integration
- ✅ Admin Dashboard
- ✅ Configurable Thresholds

## Installation

1. Upload the plugin to `/wp-content/plugins/mrx-ai-cod-fraud-protection`
2. Activate through WordPress admin
3. Configure settings in WooCommerce > COD Fraud Protection

## Requirements

- WordPress 5.8+
- WooCommerce 5.0+
- PHP 7.4+
- Dokan (optional)
